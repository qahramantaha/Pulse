## Project Scope

Pulse is an autonomous reliability platform for monitoring managed services, detecting failures, applying automated recovery actions, and tracking incidents through a central dashboard.

## Managed Services
- auth-service
- ticket-service
- notification-service

## Platform Services
- monitor-service
- healing-engine
- incident-service
- dashboard

## Version 1 Scope
- managed services with health endpoints
- monitor-service health checks
- healing-engine recovery actions
- incident-service incident tracking
- dashboard visibility
- PostgreSQL database
- Docker-based local environment
